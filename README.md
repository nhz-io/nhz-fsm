## nhz-fsm

## Install
```
npm install --save nhz-fsm
```

## Browser

* [nhz-fsm.min.js][fsm-browser-dist]
* [nhz-fsm-generator.min.js][generator-browser-dist]

## Build
```
git clone https://github.com/nhz-io/nhz-fsm.git
cd nhz-fsm
npm install
npm start
...
check http://localhost:9000
...
npm dist
```
